Protección de Datos Personales

  1. Introduccion
    Registrandose con www.elligue.com para los servicios ofrecidos, el usuario conviene usar los servicio exclusivamente apegandose a los Terminos de Servicio reconoce las siguientes reglas con respecto al uso de sus datos personales.
  2. Protección de datos personales
    Cualquier dato personal que un miembro registra en su cuenta de elligue.com se proporciona de forma voluntaria por el miembro. Los datos sobre el miembro, guardados por elligue.com, se utilizan única y exclusivamente para poder proporcionarle el servicio de elligue.com al miembro. Para este motivo, se publican y son públicamente accesibles en varias partes del sitio de elligue.com y el usuario lo reconoce y se declara de acuerdo. Solamente los datos de "contraseña", "correos electrónicos", y "fecha de nacimiento" no se dan a terceras partes, ni son accesibles legalmente a terceras partes. Sin embargo, el usuario reconoce y está de acuerdo que ningun sistema de almacenaje de datos ofrece una seguridad absoluta, y que sus datos, incluyendo los datos de "contraseña", "correos electrónicos", y "fecha de nacimiento", podrían ser apropiados por terceros a través de actos criminales, por errores de programación, o por accidentes. De cualquier forma, el usuario no responsabilizará el el dueño de elligue.com de tal pérdida de datos. El usuario declara que no almacenará datos en elligue.com que quiere que queden privados completamente. También, el usuario se declara de acuerdo que todos los datos que proporciona a elligue.com se pueden publicar en el sitio de elligue.com por tiempo indefinido. Para tal, el usuario otorga al dueño de elligue.com una licencia gratuíta e irrevocable de publicar y almacenar por tiempo illimitado los datos que el usuario proporciona.
  3. Requerimiento de datos personales
    No es necesario, ni es recomendado, proporcionar datos personales a elligue.com por los que otros pueden determinar la identidad real del usuario. El servicio se puede utilizar de forma anónimo.
  4. Access logging
    En el momento de accesar a elligue.com se transmiten ciertas datos técnicos automaticamente entre la computadora del usuario y la computadora que contiene elligue.com. Estos datos, incluyendo el número IP y el tipos de systema operativo del usuario, son guardados automatiamente por el software (web server) que facilita el acceso del software de elligue.com a la internet. Estos datos se guardan por tiempo indefinido y se utilizan para usos estadísticos.
